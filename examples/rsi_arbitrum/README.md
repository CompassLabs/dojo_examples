## Relative Strength Index Strategy

See our [docs](https://www.compasslabs.ai/docs/examples/rsi) for details.

```
ARBITRUM_RPC_URL=https://arbitrum-mainnet.infura.io/v3/<YOUR_INFURA_KEY>
LICENSE_KEY=<YOUR LICENSE KEY>
```
